# ЧМ-2026 · Семейный конкурс прогнозов

Статический сайт семейного конкурса прогнозов на чемпионат мира 2026.
Живая страница: https://alexandermoskovkin.github.io/wc2026-konkurs/

## Как устроено

- Источник истины — лист «Прогнозы» в `WC2026_konkurs.xls`
  (`C:\Users\Alex\Claude\Scheduled\wc2026-family-contest-daily\`). Файл только читается.
- `site.py` читает xls (+ `reviews.json`, если есть рядом с xls), пересчитывает очки
  и пишет `data.json`.
- `index.html` — вся страница, vanilla JS, грузит `data.json` без кэша.
- `update_site.bat` → `update_site.ps1`: генерация + git commit/push, если есть изменения.
  Запускается планировщиком Windows ежедневно в 10:30 (задача `wc2026-site-update`).
- Матчи плей-офф подхватываются автоматически: новые строки в листе «Прогнозы»
  попадают в `data.json` без правок кода.

## Очки

Точный счёт = 5. Иначе: верная разница мячей (включая ничью) = 3, верный исход = 2,
иначе 0; бонус +1 за точное число голов хотя бы одной из команд (максимум без
точного счёта = 4). Нет прогноза — очки не начисляются. Тесты: `python test_scoring.py`.

## Обзоры матчей (reviews.json)

Файл кладётся рядом с xls: `...\wc2026-family-contest-daily\reviews.json`, формат:

```json
{
  "11.06": {
    "1": "Мексика уверенно открыла турнир — 2:0, трое угадали точный счёт."
  }
}
```

Ключ верхнего уровня — дата матча (ДД.ММ), внутри — № матча из листа «Прогнозы».
